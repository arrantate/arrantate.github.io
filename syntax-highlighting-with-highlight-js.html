highlight.js

By implementing a script called highlight.js we can style any code blocks within our website which are contained in <pre><code></code></pre> tags. The website for highlight.js tells us how to do this by copying and pasting the following into out .html file:

<link rel="stylesheet" href="/path/to/styles/default.css">
<script src="/path/to/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

This is great! However you'll notice that you need to change the paths to find the script and the stylesheet. There is an option to download the highlight.js bundle but the easier way is to use the hosted versions.

That being said, if you copy and paste this version into your .html just before the closing </body> tag instead it will work straight away:

<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/default.js/10.1.1/styles/atom-one-dark.min.css">
<script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/10.1.1/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

Styles

The defualt style in my opinion isn't really very nice. So to change this we can change the href in the stylesheet link to point to the style we want. Here is a link to all of the available styles for highlight.js. Just pick the name of the style you want and paste it into the url.

<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/default.js/10.1.1/styles/atom-one-dark.min.css">

becomes...

<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/<name of the style you want>.js/10.1.1/styles/atom-one-dark.min.css">

At the time of writing this I have settled on atom-one-dark for arrantate.co.uk. So all in all this is the code that sits in my .html:

<!-- Highlight JS -->
<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/10.1.1/styles/atom-one-dark.min.css">
<script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/10.1.1/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>
