Jupyter

Pandas is easy to use in a jupyter notebook since the data is easily viewable.

Launch a notebook:

jupyter notebook

This will open up a web browser with your notebook.

Importing the data

Importing a csv:

import pandas as pd

df = pd.read_csv('<path to csv>')

Viewing the data

Shows the first n rows (default is 5):

df.head(n)

Shows the last n rows (default is 5):

df.tail(n)

Returns a tuple showing dimensions of the dataframe (rows, cols):

df.shape

Shows the type of data in each column:

df.info()

Tells pandas how many rows and colums to show:

pd.set_option('display.max_columns', 85)
pd.set_option('display.max_rows', 85)

Create a dataframe from a dictionary

Example of a dictionary:

people = {
    "first": ["Arran", "Joe", "Steve"]
    "last": ["Tate", "Bloggs", "Jobs"]
    "email": ["arrane@email.com", "joe@email.com", "steve@email.com"]
}

Creating a dataframe:

pd.DataFrame(people)

Series vs Dataframe

A series is a list of data (one dimentional array)

A dataframe is a container for multiple series objects (two dimensional array)

Returns a series for the 'email' column:

df['email']

This notation returns the same thing but can be risky if the name of the series conflicts with a dataframe method:

df.email

Returns a count of unique values in column:

df['email'].value_counts()

Returns a dataframe of multiple columns:

df.[['last', 'email']]

Returns the index of columns:

df.columns

.iloc (location by integer)

Returns series of a row by index (i=row) (returned series has an index of column names):

df.iloc[i]

Returns dataframe of multiple rows by index (i=row, j=row):

df.iloc[[i, j]]

Returns dataframs/series of rows and columns by index (i=row, j=row, k=column, l=column):

df.iloc[i, k]
df.iloc[[i, j], k] # Also works with list of rows
df.iloc[[i, [k, l]] # Also works with list of columns
df.iloc[[i, j], [k, l]] # Also works with list of rows and columns

The same can be done with list slicing. The slice is passed in rather than a list:

df.iloc[0:2, k] # returns indexes from 0 to 2

Note that the slices are inclusive. ie. index 0, 1, and 2 are included.

.loc (location by label)

Works the same as iloc but with labels instead of integer index. (if the index happens to be an integer, then the int doesnt not need to be passed as a string)

Setting indices

This would return email addresses:

df.['email']

This returns the index of the dataframe:

df.index

This would set the email address as the index:

df.set_index('email')

Note: this will not change the df. It will just return a version of the df with the altered index. to change the df you would df.set_index('email', inplace=True)

This will reset the index:

df.reset_index(inplace=True)

The index can also be set when you create the dataframe:

df = pd.read_csv('<path to csv>', index_col='email')

Order df by index (alphabetical):

df.sort_index() 
df.sort_index(ascending=False) # Reverse alphabetical order

Again use inplace=True to commit changes to the df

Filtering Dataframes

In the below, the filt variable returns a series with a True or False value for each index.

filt = (df['email'] == 'arran@email.com')

This applies the filter (which we defined above) to the df and returns a df containing all rows that returned True from the filter:

df[filt]

To put it simply. To filter the df for a given email address and return the resulting df we would have:

filt = (df['email'] == 'arran@email.com')
df[filt]

Can also be written on one line: df[df['email'] == 'arran@email.com']

Can also be acheived using .loc using filt as the rows argument, allowing us to also choose which columns we want:

df.loc[filt, 'first']
df.loc[~filt, 'first'] # adding the ~ reverses the filter

Conditionals: & means "and" and | means "or". Used like:

filt = (df['email'] == 'arran@email.com') & (df['first'] == 'arran')
